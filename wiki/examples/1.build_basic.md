# Basic CBuild usage

::: entry

This is a very simple example that just demonstrates how to use *CBuild* for buildsystem.

It is very simple - [`cbuild_cmd_run`](DOC:cbuild_cmd_run) is used to call command built into [`cbuild_cmd_t`](DOC:cbuild_cmd_t) as `cmd` variable. By default, that function also waits for command to finish before returning, so you can treat that code as normal synchronous code.

:::

::: entry

### Code

Directory structure:
```{.tree dir=examples/build/1.basic}
```

`main.c` - Your app's source code.
```{.c .lineNumbers file=examples/build/1.basic/src/main.c}
```

`cbuild.c` - Your buildscript.
```{.c .lineNumbers file=examples/build/1.basic/cbuild.c}
```

:::
:::entry

### How to run

```bash
cc -o cbuild.run cbuild.c
./cbuild.run
./app.run
```
